# medical_nlp
This repository has the beginnings of learning steps with Amazon Comprehend Medical which  is a service of Natural Language Processing for the medical area

The folder boto3_medicalcomprehen is the first attempt to get the first flavors with this API using python (named as Boto3 into AWS ecosystem). In order to accomplish, I had to carry out the following [steps](https://docs.aws.amazon.com/comprehend-medical/latest/dev/comprehendmedical-gettingstarted.html):
 1. Setup an AWS Account
 2. Setup the AWS CLI
 3. Create the python program
